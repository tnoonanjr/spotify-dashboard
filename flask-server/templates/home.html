Track your spotify footprint

Dashboard is a web app made using the Spotify API providng stats on your music profile! You can view your playlists and current top tracks. This web app was made with Flask backend using Python and html/css with JavaScript frontend dynamically-generated effects.


Log in with Spotify here to view your profile!


More about the app
This app's back-end is powered by Flask and written in Python. The website is deployed using Heroku. The front-end is written in HTML/CSS. I decided against using CSS frameworks or HTML templates. In order to use this project to learn how to design pages completely on my own. Unfortunately, the main pages are not available unless you have a Spotify account (free or premium) and agree to the terms.
